# student-report-card-system-csv
📌 Project Overview

This is a Python-based console application developed using Object-Oriented Programming (OOP). The system manages student records, generates report cards, and stores data in a CSV file for persistence.

⚙️ Technologies Used
Python
Object-Oriented Programming (OOP)
Built-in Data Structures (List, Dictionary)
CSV File Handling
🧠 Features
Add multiple students with details
Store subject-wise marks
Calculate total marks and percentage
Generate grade automatically
Display report cards
Save student data to CSV file
🏗️ OOP Concept Used
Encapsulation
▶️ How to Run
Run the Python file (report_card.py)
Enter student details
View report card
Data will be saved automatically in CSV file
📈 Output
Displays student report card
Generates percentage and grade
Saves data in students_data.csv file
